# Machine-Learning-Classification-Project
## Overview

This project implements and evaluates multiple machine learning classification algorithms using Python and Scikit-learn.

It includes model training, comparison, visualization, and evaluation using both a toy dataset (PlayTennis) and a real-world dataset from UCI repository.


## Objectives

- Implement Decision Tree classifier
- Compare multiple ML models (KNN, Naive Bayes)
- Apply models on UCI dataset
- Evaluate performance using metrics
- Visualize results and decision boundaries

## Project Structure

- **Part A:** Decision Tree on PlayTennis dataset
- **Part B:** Classifier comparison (KNN vs Naive Bayes)
- **Part C:** UCI dataset classification and evaluation


## Part A: Decision Tree

- Dataset: PlayTennis
- Model: Decision Tree Classifier
- Output:
  - Tree visualization
  - Model accuracy

## 🟡 Part B: Classifier Comparison

Models used:
- K-Nearest Neighbors (KNN)
- Naive Bayes

### Evaluation:
- Accuracy comparison
- Performance analysis
- Visualization plot

## Part C: UCI Dataset

- Dataset: Selected from UCI Repository
- Train/Test Split: 80/20

### Metrics:
- Accuracy
- Confusion Matrix
- Precision
- Recall
- F1-score

## Results Summary

- Decision Tree: Good interpretability
- Naive Bayes: Fast but less flexible
- KNN: Strong performance with tuning
